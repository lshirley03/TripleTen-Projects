Superstore Analysis Project



This project analyzes the Superstore dataset to identify profit centers, loss-makers, and return patterns to help save the superstore from bankruptcy.



\## Tableau Public Dashboard



You can view the interactive dashboard here: [Sprint 4 Project](https://public.tableau.com/views/Sprint4Project_17594715525250/RegionSub-Category?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)



\## Project Overview



This analysis covers:

\- Part 1: Profits \& Losses analysis

\- Part 2: Advertising recommendations  

\- Part 3: Returned items analysis
