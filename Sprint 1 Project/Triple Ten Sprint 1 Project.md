# Triple Ten Sprint 1 Project

#Title: Manhattan Rental Property Analysis

## Project Overview
This project provided a client with a data-driven investment strategy for the Manhattan vacation rental market. 
It involved cleaning a large Airbnb dataset and using pivot tables to identify the most attractive neighborhoods, property sizes, and revenue drivers for optimal investment.

## Key Findings
Optimal Investment: Midtown studios generate the highest revenue of any neighborhood-property type combination.

Market Demand: Identified the Top 10 neighborhoods and confirmed property size popularity (e.g., 1-bedroom units dominate 50% of the Harlem market).

Strategic Validation: Calculated revenue generated by top listings to provide a confident, data-backed investment strategy.
