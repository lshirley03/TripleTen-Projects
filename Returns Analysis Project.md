\# Returns Analysis Dashboard



\## Project Overview

This dashboard analyzes return patterns to identify root causes and business opportunities.



\## Dashboard Link

View the interactive dashboard here: [Sprint 5 Project](https://public.tableau.com/views/Sprint5Project_17607375970730/CategorialReturnRate?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)



\## Key Findings

\- Our analysis indicates that returns have a weak correlation to overall sales, proving issues are specific, not general. Our Tableau Dashboard and story help us find the root causes. We then create an action plan to solve the root issues.
